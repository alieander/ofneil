# Thomas O'Neil Resume Site

Single-page personal resume site built as static HTML and CSS.

## Stack

- HTML
- Custom CSS in [css/main.css](css/main.css)
- Bootstrap 5.3.3 via CDN for modal behavior and baseline component styling

## Files

- [index.html](index.html): page markup
- [css/main.css](css/main.css): site theme and layout
- [tommy-resume.pdf](tommy-resume.pdf): downloadable resume

## Notes

- The page no longer uses local Bootstrap or jQuery assets.
- The contact dialog depends on the Bootstrap bundle loaded from CDN in [index.html](index.html).
- Print styles are included in [css/main.css](css/main.css) so the page can also be used as a printable resume.
